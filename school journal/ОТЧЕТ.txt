## Точный отчет по проекту: Система Учета Успеваемости Студентов (Grade Management System)

Проект представляет собой мини-приложение для управления данными студентов и их оценками, построенное на фреймворке Flask с использованием объектно-ориентированного подхода (ООП) для моделирования данных.

### 1. Архитектура и Технологии

| Аспект | Детали реализации | Файл |
| :--- | :--- | :--- |
| **Основной Фреймворк** | Flask | `app.py` |
| **Хранение данных** | Локальный JSON-файл (`data.json`) | `app.py`, `data.json` |
| **Объектная модель** | Классы `Student`, `Subject`, `Grade` | `models.py` |
| **Инициализация** | Приложение инициализирует 4 предмета и 6 студентов по умолчанию, если файл `data.json` не найден или пуст/некорректен | `app.py` |

### 2. Объектно-Ориентированная Модель (`models.py`)

Система построена на трех основных классах, реализующих базовые принципы ООП:

#### А. Класс `Subject` (Предмет)
* **Назначение:** Представление учебной дисциплины.
* **Поля:** `name` (название), `icon` (иконка для фронтенда, например, "fa-solid fa-calculator").
* **Текущие предметы:** "Математика", "Литература", "Физика", "История".

#### Б. Класс `Grade` (Оценка)
* **Назначение:** Представление конкретной оценки.
* **Поля:** `value` (значение оценки, int), `subject` (ссылка на объект `Subject`).
* **Проверка:** Требует, чтобы переданный предмет был объектом класса `Subject`.

#### В. Класс `Student` (Ученик)
* **Назначение:** Представление ученика.
* **Поля:** Уникальный `id` (генерируется с помощью статического счетчика `_id_counter`), `name` (имя ученика), `grades` (список объектов `Grade`).
* **Методы:**
    * `add_grade(grade)`: Добавляет объект `Grade` в список оценок.
    * `get_average_grade()`: Рассчитывает средний балл ученика. Возвращает `0.0`, если оценок нет.

### 3. Функциональность Приложения (Маршруты Flask)

Приложение `app.py` определяет набор маршрутов для полного цикла управления данными учеников (CRUD - Create, Read, Update, Delete):

| Маршрут (URL) | Метод | Функция Flask | Описание Функциональности |
| :--- | :--- | :--- | :--- |
| **`/`** | `GET` | `index` | Главная страница. Отображает список всех учеников из `STUDENTS_DB`. |
| **`/add_student`** | `GET/POST` | `add_student` | **C**reate. Отображает форму и обрабатывает ее: создает новый объект `Student`, добавляет его в базу (`STUDENTS_DB`) и сохраняет данные. |
| **`/student/<int:id>`** | `GET` | `student_details` | **R**ead. Страница с детальной информацией об ученике и его оценках. |
| **`/student/<int:id>/add_grade`** | `POST` | `add_grade` | **U**pdate. Обрабатывает форму добавления оценки, добавляет объект `Grade` ученику и сохраняет данные. |
| **`/student/<int:id>/delete`** | `DELETE` | `delete_student` | **D**elete. Удаляет ученика из базы по его ID и сохраняет данные. |
| **`/student/<int:id>/report`** | `GET` | `generate_report` | **Отчет.** Отдельная страница для генерации отчета для родителей по ученику. |
